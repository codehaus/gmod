Images

imageDraws an image on the specified coordinates (x,y). You will need to specify one of the following properties to locate the image
  • image: must a BufferedImage
  • file: a filename (absolute)
  • url: an URL like http://groovy.codehaus.org/download/attachments/1866/groovydukemed.jpg
  • classpath: a resource available in the current classpath
properties [[image|file|url|classpath],x,y]
